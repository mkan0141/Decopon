# オセロAI Decopon
## Decoponについて
Decoponは発展途上中のオセロAIです。  
名前の由来は「開発当時、実家で一番取れていた果物だったから」です。つまり特に意味はありません。  

## 使い方
```
# インストール
$ git clone https://github.com/mkan0141/Decopon

# 実行
$ make 
$ game
```
